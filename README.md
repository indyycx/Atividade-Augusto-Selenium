Automação Web com Selenium – Python

Sobre o projeto
Este projeto foi desenvolvido como atividade prática para a disciplina, com o objetivo de aprender e praticar automação web utilizando Selenium em Python.
O script acessa uma página da Wikipédia, extrai informações básicas do conteúdo e executa algumas ações de navegação e teclado, simulando a interação de um usuário real com o navegador.

Tecnologias utilizadas
-Python 3
-Selenium
-Google Chrome
-WebDriver (ChromeDriver)

O código desenvolvido tem como objetivo demonstrar o uso do Selenium para automação web em Python.
Ele abre o navegador Google Chrome e acessa uma página da Wikipédia sobre a linguagem Python.
Em seguida, o programa coleta o título principal da página e o primeiro parágrafo do conteúdo, exibindo essas informações no terminal.
Também são realizadas algumas interações básicas com o navegador, como o uso de teclas do teclado para simular ações do usuário.
Além disso, o código utiliza um event listener para acompanhar eventos de navegação, retornando à página anterior antes de encerrar a execução.
